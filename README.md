# How to start
1. `npm install`
2. `node index.js`

# Deploy:
1. install nvm: `https://docs.aws.amazon.com/sdk-for-javascript/v2/developer-guide/setting-up-node-on-ec2-instance.html`
`curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.32.0/install.sh | bash`
2. install node
